# 自动回复

1. 使用 `boss list --unread` 获取未读会话。
2. 逐个执行 `boss chat "<姓名>" --strict`，读取完整可见对话。
3. 回复前校验 2027 届、最终学历目标学校、理工科、本科或研究生；任一不符合或无法识别都不回复。
4. 检测用户手动发送的新消息；存在时标记 `manual_takeover`，等待候选人再次回复。
5. 候选人请求附件简历时按 `candidate_conversion.md` 接收并验证。
6. 候选人问机会时使用 `followups.md` 的固定首问或重复问分支。
7. 候选人谈投递、转投、测评、笔试或面试时，提取结构化字段并调用 `evaluate-transfer`。
8. 普通问题只使用已安装 references 中明确的答案。
9. 知识库没有答案时，不发送；记录 `knowledge_gap` 的脱敏摘要并提示用户。
10. 每次成功动作后记录对应事件，失败动作不计数。

交换微信、附件接收、投递终止等状态动作后必须重新读取会话验证。
