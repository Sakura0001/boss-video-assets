## 候选人询问不同岗位入口区别

不同岗位入口主要会影响笔试题型和分数线。软件开发一般是 3 道编程大题，分数线一般 150 分；AI 应用开发一般是 2 道大题加一些选择题，分数线一般 200 分；网络安全方向主要是网安相关题。更细的题型可以去小红书、微信公众号或者网上经验贴找一下参考参考，公司内部招聘流程都差不多。

## 投递状态确认后邀请微信沟通

使用前置条件：

- 候选人已经回复并表现出意向。
- 已确认候选人还没有投递，或投递的不是云软件研发部/合适 ICT 部门，或候选人明确希望我帮忙跟流程。
- 不要在刚打招呼、只补充岗位介绍、还没确认投递状态时执行 `boss action wechat`。

同学，沟通下来我感觉你和我们数据库方向还挺匹配的。如果你方便的话，可以加我微信，后续沟通会更及时一些，我也可以继续帮你跟一下流程。

推荐 CLI：

```bash
boss send --text "同学，沟通下来我感觉你和我们数据库方向还挺匹配的。如果你方便的话，可以加我微信，后续沟通会更及时一些，我也可以继续帮你跟一下流程。"
```
