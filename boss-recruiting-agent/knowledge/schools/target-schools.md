# 目标学校名单

## 使用方式

本文件用于维护 Boss 自动招聘 Agent 的目标学校名单。Agent 在主动打招呼前，应先识别候选人的最终学历学校，并根据本文件判断是否继续触达。

当前知乎链接正文无法被自动抓取，请将链接中的学校名单粘贴到本文件对应分类下。

## 匹配规则

- 以候选人最终学历学校为准。
- 学校名称应尽量使用官方中文全称。
- 可在“别名”中维护常见简称，例如“北航”对应“北京航空航天大学”。
- 命中任一目标学校或别名，视为通过学校筛选。
- 未识别学校或学校不在名单内，默认不主动打招呼。
- 不要把候选人个人信息写入本文件。

## 自动化处理口径

### 学校在名单中

允许进入主动打招呼流程，并结合岗位知识库生成沟通话术。

### 学校不在名单中

默认跳过主动打招呼，不向候选人发送与学校筛选相关的解释。

### 学校无法识别

默认跳过主动打招呼；如后续需要更激进策略，可在 `config/school_policy.yaml` 中改为人工审核。

## 维护注意事项

- 本名单仅作内部招聘匹配策略使用。
- 不要在对候选人的自动回复中暴露学校筛选逻辑。
- 如名单来自内部政策，应保持版本更新，避免使用过期名单。
